# AtCoderBigStandings
## はじめに
 これはAtCoderの順位表をカスタマイズして表示する非公式UserScriptです。 FirefoxのGreaseMonkeyとGoogleChromeのTamperMonkeyで動作確認しています。 javascript素人が書いたので予期せぬバグや、AtCoder側の仕様変更により使えなくなる場合があるかも知れません。 大切なコンテストでの使用は自己責任でお願いします。
